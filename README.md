# MNSSA Web

Modern MNSSA website built with HTML, CSS, vanilla JavaScript, Supabase as backend, and GSAP for animations.

## Getting started

1. Create a new Supabase project.
2. In `js/supabaseClient.js`, replace `SUPABASE_URL` and `SUPABASE_ANON_KEY` with your project credentials.
3. Create tables: `teams`, `members`, `events`, `forms`, `form_fields`, `form_responses`, `form_answers`, `news`.
4. Serve the site using a simple HTTP server (for example `npx serve .` from the project root).
